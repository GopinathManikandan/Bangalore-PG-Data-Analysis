#  Bangalore PG Accommodation Analytics (2025–2027)

##  Project Overview
This project analyzes PG accommodation data in Bangalore 
city using Python, Pandas, NumPy and Visualization libraries.

##  Dataset
- 1080 rows and 14 columns
- Covers PG listings across Bangalore (2025–2027)

##  Tools Used
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Jupyter Notebook

##  What I Did
- Cleaned messy real-world dataset
- Fixed missing values, duplicates, inconsistent data
- Feature Engineering
- Descriptive Statistics
- Answered 25 questions using Pandas, NumPy & Visualization

##  Key Findings
- Koramangala has highest PG listings in Bangalore
- Single rooms are most preferred (31%)
- Most PGs are affordable (below ₹10,000/month)
- Metro distance does NOT affect PG rent
- 40% of PGs are above average rating
